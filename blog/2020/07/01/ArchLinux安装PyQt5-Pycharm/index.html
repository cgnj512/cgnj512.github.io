Pycharm 中配置 Designer 和 PyUIC 快捷启动

打开 File -> Settings -> Tools -> External Tools ，新建：

  • Qt-Designer

    Program 填入 /usr/bin/designer，Working Directory 填入 $FileDir$。

  • PyUIC

    Program 填入 /usr/bin/python3 ，

    Arguments 填入 -m PyQt5.uic.pyuic $FileName$ -o $FileNameWithoutExtension$.py，

    Working Directory 填入 $FileDir$。
